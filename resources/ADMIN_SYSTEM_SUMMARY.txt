🎯 THREE WAYS TO ADD QUESTIONS
════════════════════════════════════════════════════════════════════════════

METHOD 1: ADMIN DASHBOARD (/admin)
─────────────────────────────────
For:          Adding 1-5 questions
Time:         2-10 minutes
Difficulty:   Easy
Steps:
  1. Navigate to /admin
  2. Fill question form:
     • Question text
     • Topic (dropdown)
     • Difficulty (Easy/Medium/Hard)
     • Companies (checkboxes)
  3. Click "Add Question"
  4. See success message
  5. JSON appears in browser console
  6. Copy JSON
  7. Paste into companyQuestions.ts
  8. Done!

Features:
  ✅ Visual form
  ✅ Company selection checkboxes
  ✅ Real-time validation
  ✅ View all questions
  ✅ Export to CSV/JSON
  ✅ Import from files
  ✅ Status messages


METHOD 2: CSV IMPORT/EXPORT
──────────────────────────
For:          Bulk adding 5-50 questions
Time:         30-60 minutes
Difficulty:   Medium
Steps:
  1. Go to /admin → Export tab
  2. Download CSV
  3. Open in Excel/Google Sheets
  4. Add new rows with questions
  5. Save as CSV
  6. Go to /admin → Import tab
  7. Upload your CSV
  8. Copy JSON from console
  9. Paste into companyQuestions.ts
  10. Done!

Format Example:
  Question,Topic,Difficulty,Companies
  "Your question here",Arrays,Medium,"Google, Meta, Amazon"


METHOD 3: DIRECT JSON EDITING
────────────────────────────
For:          Developers, quick edits
Time:         5-10 minutes
Difficulty:   Medium (need JSON knowledge)
Steps:
  1. Open src/data/companyQuestions.ts
  2. Find your topic section
  3. Add new question object:
     {
       text: "Your question",
       topic: "Topic Name",
       difficulty: "Medium",
       companies: ["Google", "Meta"]
     }
  4. Save file
  5. Test at /company
  6. Done!

💾 DATA STRUCTURE REFERENCE
════════════════════════════════════════════════════════════════════════════

Each question has these fields:

{
  "text": "Your interview question here",        // REQUIRED: string
  "topic": "Arrays",                             // REQUIRED: string
  "difficulty": "Medium",                        // REQUIRED: Easy|Medium|Hard
  "companies": ["Google", "Meta", "Amazon"]     // REQUIRED: string[]
}

Example:
{
  "text": "How do you reverse a linked list?",
  "topic": "Linked Lists",
  "difficulty": "Medium",
  "companies": ["Google", "Meta", "Amazon", "Apple"]
}

⚙️ ADMIN DASHBOARD TABS EXPLAINED
════════════════════════════════════════════════════════════════════════════

TAB 1: ➕ ADD QUESTION
  Purpose:     Add new questions one at a time
  Form Fields: Text, Topic, Difficulty, Companies
  Output:      JSON in console
  Action:      Click "Add Question"
  Result:      Stored in localStorage + console log

TAB 2: 👁️ VIEW ALL
  Purpose:     See all 100+ existing questions
  Display:     List format with color coding
  Filter:      None (shows all)
  Use Case:    Verify questions, check duplicates

TAB 3: 📥 EXPORT
  Purpose:     Download all questions
  Formats:     CSV or JSON
  Action:      Click "Download CSV/JSON"
  Use Case:    Backup, bulk edit, team sharing

TAB 4: 📤 IMPORT
  Purpose:     Upload CSV/JSON files
  Formats:     .csv or .json
  Action:      Click file input, select file
  Output:      JSON in console for copy/paste
  Use Case:    Bulk add, team collaboration
