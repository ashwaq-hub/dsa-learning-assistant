SCENARIO 1: Adding a Question
  1. Go to /admin
  2. Click "Add Question"

  4. Fill form
  5. Click "Add Question"
  ✅ Done!

SCENARIO 2: Practicing Questions
  1. Go to /company
  2. Select company (e.g., Google)
  3. Click "SOLID Principles" tab
  4. Browse/expand questions
  ✅ Start learning!

SCENARIO 3: Quiz Mode
  1. Go to /company
  2. Select company
  3. Click "Quiz Mode"
  4. New topics automatically included
  ✅ Practice with progress bar!

SCENARIO 4: Bulk Add
  1. Go to /admin
  2. Export → Download CSV
  3. Edit in Excel (add new rows with SOLID/OOP/OOD)
  4. Save CSV
  5. Import back via /admin
  ✅ Bulk added!
